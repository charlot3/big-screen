BIG · Shortcuts & features

Show one value, enormous. Clock, countdown & stopwatch; themes, rainbow, scroll ticker, shareable links — all in one file, fully offline.

Keyboard

Enter
Show the typed value (Text mode)
Esc
Clear value · or close this help / color drawer
Space
Start / pause the countdown or stopwatch (when not typing)
F
Toggle fullscreen
S
Toggle scroll / ticker mode
R
Toggle rainbow / unicorn color mode
M
Mystery — apply a random color treatment
C
Toggle the color & theme drawer
H
Hide / show all controls for a clean display
Q
Show a scannable QR code of this view
?
Toggle this help overlay
any key
Reveals the controls · jumps focus to the value field (Text mode)
tap stage
Tap/click empty screen area to show or hide the controls

The controls auto-hide after a few idle seconds for a clean screen. Move the mouse, tap, press any key, or hit H to bring them back — you are never left without a way in.

Modes

Text
Type anything, or pick a preset. Turn on Cycle to rotate through multiple lines.
Clock
Live local time. Toggle 12/24-hour and seconds.
Countdown
Set mm:ss / hh:mm:ss, or a target clock time. Start, pause, reset, optional loop; it flashes & reads DONE at zero.
Stopwatch
Counts up from zero. Start, pause, reset.

More

Size
Scale the displayed value from 20% up to the auto-fit fill (100%). Works in every mode.
Mystery
One tap (or M) rolls a random, pleasing color treatment — a theme, a rainbow palette, or a custom pair.
Hide
The eye button or H hides the dock for a clean display; controls also fade out when idle.
Share
Copies a link that restores this exact view (mode, value, theme, colors, rainbow, scroll, size, cycle).
QR
Shows a scannable QR code of the share link — point a phone camera at the big screen to open this exact view. Fully offline, no QR service.
Themes / DIY
10 presets plus custom text/background colors and a live rainbow palette.
Install
Add to Home Screen to launch it app-like; it runs fully offline.
